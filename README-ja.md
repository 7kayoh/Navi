# Navi
Navi は、Fusion が持つ独特な宣言型デザインを活用したシンプルなルーティングライブラリです。これにより、フロントエンド開発初心者でも学習しやすい学習曲線を提供します。

ただし、Navi はまだベータ版であり、多くの高度な機能はまだ欠落しており、後続バージョンに追加されない可能性があります。そのような場合は、より高度なバージョンである Koute の使用を検討してください。
